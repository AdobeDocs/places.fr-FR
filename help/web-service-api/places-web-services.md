# Présentation de l’API Web Services {#places-web-services-api}

Le service Places est l’ensemble de services qui permet aux clients d’Adobe d’intégrer plus facilement les solutions Adobe Cloud Platform et Adobe Experience Platform aux données de localisation et à l’expérience appropriée pour la bonne personne au bon moment et au bon endroit.

Les API de services web vous permettent d’effectuer les opérations suivantes :

* Gestion des limites géographiques
* Mesurer la position des utilisateurs même lorsque l’application est en arrière-plan
* Utiliser les données en temps réel quand c’est important

Cette section fournit des informations sur l’utilisation des API REST et de la base de données des points d’intérêt, qui contient les données des points d’intérêt de votre organisation.

## API REST

L’API REST Places Service vous permet de travailler par programmation avec les points d’intérêt de votre entreprise. Ces API vous permettent de créer, mettre à jour et supprimer vos bibliothèques et les points d’intérêt dans ces bibliothèques. Ces API utilisent les normes JavaScript Object Notation (JSON) pour formater les données envoyées et reçues. L’un des principaux avantages de JSON est qu’il rend les requêtes d’API faciles à écrire, lire et analyser par les développeurs et les machines.

Avant de pouvoir utiliser l’API de services web, assurez-vous que les conditions suivantes ont été remplies :

* Le service Places est configuré dans votre organisation et vous disposez d’un accès approprié en tant qu’utilisateur.

  Pour plus d’informations, consultez *Conditions préalables pour l’accès utilisateur* dans [Présentation de l’intégration et conditions préalables](/help/web-service-api/adobe-i-o-integration.md).

* Une fois que le service Places est configuré dans votre organisation et que vous y avez accès, créez une intégration Adobe pour le service Places.

  Pour plus d’informations, voir *Création d’une intégration Places Service* dans [Présentation et conditions préalables de l’intégration](/help/web-service-api/adobe-i-o-integration.md).

Informations supplémentaires :

* Pour plus d’informations sur les API disponibles et leur utilisation, voir [Gestion des bibliothèques](/help/web-service-api/api-usage/manage-libraries/manage-libraries.md) et [Gestion des points d’intérêt](/help/web-service-api/api-usage/manage-pois/manage-pois.md).
* Pour plus d’informations sur les en-têtes et les paramètres dans ces API, voir [En-têtes et paramètres](/help/web-service-api/api-usage/headers-and-parameters.md).
